# Upstream Bootstrap Snapshots

This folder manages static snapshots representing upstream bootstrap/install scripts.

- `asahi-bootstrap-dev.snapshot.sh`: Mock layout for Asahi bootstrap variables setup.
- `asahi-install-sh.snapshot.sh`: Mock layout for install script entrypoint.
- `alarm-installer-bootstrap.snapshot.sh`: Mock layout for Arch Linux ARM specific installer bootstrap setup.
- `alarm-install-sh.snapshot.sh`: Mock layout for ALARM specific install script entrypoint.
