# Static Bootstrap Override Inspection Workspace

This folder manages script tools and snapshots for analyzing upstream Asahi/ALARM bootstrap script variables and partition boundaries.

---

## 1. Directory Structure

- `upstream/`: Static snapshots of upstream bootstrap scripts.
- `notes/`: Variables maps and boundaries analysis logs.
- `fixtures/`: Mock fixtures containing partition/danger rules.

---

## 2. Safety Guidelines

> [!WARNING]
> Do not execute upstream bootstrap scripts from this directory.
> These files are static snapshots for inspection only.
