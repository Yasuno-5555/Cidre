# Cidre v0.32.0: Cidre.app Prototype Pack

## Summary
Cidre v0.32.0 introduces the first macOS application prototype (`Cidre.app`).

## Motivation
To support future GUI onboarding and uninstall, it is essential to build the initial SwiftUI interface layout and backend metadata loaders.

## Added layouts
- `apps/macos/CidreApp/`
- SwiftUI dashboard, install, uninstall, repair, settings, and reports views.
- Mock fixtures for manifest, actions, command results, and reports.

## Safety Model
- Read-only prototype execution limits.
- Blocked destructive actions in GUI.
- Sudo/root verification limits.
