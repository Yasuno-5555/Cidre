# macOS Guided Uninstall

- **macOS Dashboard**: Show guided uninstaller status from macOS.
- **Disk Audit**: Scan and view storage layout from the macOS perspective using `diskutil`.
- **Target Scan**: Safely classify target partitions before any modification.
- **Dry-run**: Mock macOS target alterations.
- **Restore Report**: Produce a final confirmation document of the cleanup status.
