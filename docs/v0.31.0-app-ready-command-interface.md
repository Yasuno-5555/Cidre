# Cidre v0.31.0: App-Ready Command Interface Pack

## Summary
Cidre v0.31.0 standardizes backend interfaces for GUI integration (specifically `Cidre.app`).

## Motivation
To decouple frontend and backend and prevent scraping of CLI logs, CLI tools are enhanced to return structured, schema-compliant JSON data, standardize exit codes, support plain/no-color outputs, and support noninteractive executions.
