# Cidre v0.30.0: Fully Guided Uninstaller Pack

## Summary
Cidre v0.30.0 introduces a guided uninstall flow. It enables safe removal preparation of Cidre/Asahi installations and transitions the system back to macOS.

## Motivation
With a guided installation flow in v0.29.0, it is critical to offer a symmetrical guided exit path in v0.30.0 to prevent user confusion and lower the risks of accidental partition deletion.

1. `uninstall-preflight`
2. `state-export`
3. `partition-audit`
4. `target-scan`
5. `target-review`
6. `risk-check`
7. `dry-run`
8. `macos-restore-handoff`
9. `execute-optional`
10. `post-uninstall-verify`
11. `uninstall-complete`

## Safety Model
- Mandatory state export prior to planning/destructive dry-run.
- Default to dry-run (read-only mode).
- Block protected volumes (macOS system, data, recovery).
- Require exact target specification; automatic target guessing is strictly prohibited.
