# Cidre v0.33.0: Cidre.app Guided Action UI Pack

## Summary
Cidre v0.33.0 extends the macOS SwiftUI prototype by adding support for safe read-only process execution, repository path selection, safety validation against manifest constraints, execution log views, and markdown report displays.

## Motivation
With a valid CLI interface contract finalized in v0.31.0 and view skeletons prepared in v0.32.0, this release wires the visual panels with real read-only processes and policy checkers, ensuring the prototype behaves like a guided tool without risking system modifications.

## Added Layout
* ViewModels and Services for workspace picking and process execution.
* Custom UI validation, safe action listings, and mock generation scripts.

## Safety Model
* Refuses destructive actions statically and dynamically.
* Blocks root privilege elevation requirements.
* Promotes validation visibility for future integration tasks.
