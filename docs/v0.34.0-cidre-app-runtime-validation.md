# Cidre v0.34.0: Cidre.app Runtime Validation Pack

## Summary

v0.34.0 validates that the macOS SwiftUI prototype can be built, launch-prepared, and exercised with safe read-only guided actions.

## Motivation

v0.32.0 introduced the app shell and v0.33.0 introduced guided actions. v0.34.0 connects those pieces to real macOS runtime checks.

## Added Scripts

- `scripts/cidre-app-runtime-check`
- `scripts/cidre-app-build-check`
- `scripts/cidre-app-launch-check`
- `scripts/cidre-app-runtime-report`
- `scripts/cidre-app-runtime-clean`
- `scripts/cidre-app-runtime-fixture`
- `scripts/cidre-app-runtime-log-collect`
- `scripts/cidre-app-runtime-result`

## Updated Components

- Runtime status is exposed in the dashboard.
- Repository validation now includes the macOS app directory.
- Safe command execution shows blocked actions and JSON parse failures more clearly.
- `cidre-doctor` and `cidre-recovery` now expose app runtime validation entry points.

## Non-goals

v0.34.0 does not implement privileged helper support, real install, real uninstall, APFS mutation, or notarized packaging.
