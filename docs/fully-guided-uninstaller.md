# Fully Guided Uninstaller

- **Purpose**: Safe exit path from Cidre to macOS.
- **Stages**: Symmetrical to guided install stages.
- **Dashboard**: Interactive terminal display of exit progress.
- **Mandatory Export**: Back up logs, reports, and system state before deletion.
- **Target Scan/Review**: Categorize disk partitions and review targeting safety.
- **Dry-run**: Show planned steps without making changes.
- **Report**: Generate exit report for future troubleshooting.
- **macOS Restore Handoff**: Provide guides and links for cleanup from macOS.
