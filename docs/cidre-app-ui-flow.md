# Cidre.app UI Flow

- **Sidebar selection list**: Navigation items to switch active detail view.
- **Dashboard view**: Shows overall system status and manifest command metrics.
- **Install view**: Lists installation actions.
- **Uninstall view**: Lists uninstallation preparation actions.
- **Repair view**: Lists safe diagnostics actions.
- **Reports view**: Interactively displays raw markdown reports.
- **Settings view**: Manages mockup mode toggles and repository paths.
