# Install Handoff

Handoff files are generated dynamically between phases using `cidre-install-handoff`.

## macOS to ALARM Handoff
Generates `install-handoff.md` with instructions to run Asahi Linux installer and boot fresh ALARM.

## Root to User Handoff
Saves user login steps, reminding the installer runner to login as their chosen normal user instead of root.

## User to Firstboot Handoff
Instructs the user to reboot.
