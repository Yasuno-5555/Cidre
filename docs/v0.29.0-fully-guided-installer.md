# Cidre v0.29.0: Fully Guided Installer Pack

## Summary
Cidre v0.29.0 adds structured tracking of all installer phases, dashboard outputs, and dynamic handoff documentation.

## Non-goals
- Fully automatic partition resizing
- Automated bootloader edits or boot entries.

## Verification Checklists
- Run `cidre-install-stage --validate` to verify JSON syntax.
- Verify status outputs with `cidre-install-dashboard`.
