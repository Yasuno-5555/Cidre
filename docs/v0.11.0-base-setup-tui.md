# Cidre v0.11.0 - Base Setup TUI Pack

Cidre v0.11.0 improves the fresh ALARM base setup experience by adding an interactive dashboard/wizard layer around the root-phase `./preinstall` flow.

## Highlights

- Added guided `./preinstall` mode with backend selection across `dialog`, `whiptail`, and plain shell fallback.
- Added a base setup dashboard summarizing system, network, pacman, user, sudo, tools, and next-step readiness.
- Added guided target-user selection/creation and isolated wheel/sudo setup validation.
- Added final root-to-user handoff output and root-owned repository warning.
- Added preinstall logging under `~/.local/state/cidre/preinstall/`.
- Aligned `cidre-doctor --base-readiness` wording with the root-phase flow.

## Entry Points

```bash
./preinstall
./preinstall --check
./preinstall --dry-run
./preinstall --prepare
./preinstall --tui
./preinstall --non-interactive --dry-run --prepare
```

## Scope

v0.11.0 improves the guided base setup after ALARM is already installed. It does not replace:

- ALARM installation media
- disk partitioning
- bootloader setup
- m1n1 or DTB/FDT configuration
- a full graphical installer

Cidre v1.0.0 remains reserved for a substantially simpler installation experience.
